# BREAKOUT - Atari 2600 Clone

A browser-based Breakout clone that evokes the retro Atari 2600 aesthetic.

## Play Now

Open `index.html` in your browser or play the hosted version on GitHub Pages.

## Features

- **Authentic Retro Visuals** - CRT scanlines, screen glow, chunky pixels, rainbow brick rows
- **Classic Gameplay** - 6 rows of bricks, 3 lives, ball speeds up as you progress
- **Retro Sound Effects** - Beeps and blips using Web Audio API
- **Multiple Controls** - Keyboard, mouse, and touch support for mobile devices

## Controls

| Input | Action |
|-------|--------|
| Arrow Keys / A-D | Move paddle |
| Mouse | Move paddle |
| Touch/Drag | Move paddle (mobile) |
| Space / Tap | Start game / Launch ball |
| P | Pause game |

## Scoring

| Brick Color | Points |
|-------------|--------|
| Red (top) | 7 |
| Orange | 5 |
| Yellow | 4 |
| Green | 3 |
| Blue | 2 |
| Purple (bottom) | 1 |

## Technology

- HTML5 Canvas
- Vanilla JavaScript
- Web Audio API
- No external dependencies
